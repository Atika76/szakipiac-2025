KivitelezésPRO + Építési Napló kapcsolat javítás

SzakiPiac oldali módosítások:
- kivitelezes-pro.html: Mentett projektjeim lista hozzáadva
- kivitelezes-pro.html: ügyfél saját KivitelezésPRO mentéseit látja, admin minden mentést lát
- kivitelezes-pro.html: Betöltés / Törlés / Építési Napló gombok a mentett projekteknél
- kivitelezes-pro.html: új "Mentés + Építési Napló" gomb
- send-quote-to-epitesnaplo Edge Function: már nem csak ajanlatok táblát kezel, hanem kivitelezes_projektek sort is
- send-quote-to-epitesnaplo Edge Function: nem általános non-2xx hibát dob, hanem olvasható magyar error üzenetet ad vissza

Fontos:
- A KivitelezésPRO mentés a SzakiPiac Supabase public.kivitelezes_projektek táblába ment.
- Az Építési Napló átadás a send-quote-to-epitesnaplo functionön keresztül készít import linket.
- A SzakiPiac Supabase Edge Function Secrets alatt kell: EPITESNAPLO_IMPORT_ENDPOINT és EPITESNAPLO_IMPORT_SECRET.
