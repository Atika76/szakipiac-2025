MUNKAFIGYELŐ JAVÍTÁS – OLVASD EL

Ezt javítottam:
1. A Munkafigyelő listában külön szűrő lett:
   - Összes
   - Megrendelői munkák
   - Közbeszerzések / beruházások
   - Nyilvános források

2. A SzakiLead ZIP-ben lévő közbeszerzési találatok be lettek építve.
   Ha a Supabase tábla még üres, akkor is megjelennek a Munkafigyelőben.

3. A "Munka feladása" résznél érthetőbb belépési üzenet van.
   Munkát feladni bejelentkezett felhasználóval lehet.

4. Készült egy külön SQL fájl:
   MUNKAFIGYELO_SUPABASE_SQL_FUTTASD.sql

FONTOS:
Ahhoz, hogy a munkahirdetés feladás élesben is működjön, ezt a fájlt futtasd le a Supabase-ben:

Supabase -> SQL Editor -> New query -> másold be vagy nyisd meg a fájlt -> Run

Ha ez nincs lefuttatva, akkor a felület megjelenhet, de:
- nem lehet rendesen munkát feladni
- a Supabase-ből nem jönnek a közbeszerzések
- csak a beépített minta közbeszerzések látszanak

A közbeszerzések automatikus frissítéséhez később külön Edge Function / cron kell.
Ebben a javításban a meglévő ZIP-ben lévő találatokat építettem be és a megjelenítést javítottam.
