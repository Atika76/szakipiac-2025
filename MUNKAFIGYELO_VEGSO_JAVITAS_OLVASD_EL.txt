Munkafigyelő végső javítás

Javítva:
- a Munka közzététele gomb most type=button, direkt kattintásra indít mentést
- kattintáskor azonnal látszik: Ellenőrzés… / Mentés…
- mentés előtt újra lekéri a Supabase belépési sessiont
- ha nincs session, kiírja, hogy újra be kell jelentkezni
- ha SQL/RLS/RPC hiba van, a form alatt megjelenik a pontos hiba
- bekerült a hiányzó isUsablePublicJob szűrő, így link nélküli külső találat nem jelenik meg
- frissítve az index import verziója és service worker cache neve

Kötelező Supabase lépés:
1. Supabase > SQL Editor > New query
2. Másold be és futtasd: MUNKAFIGYELO_KOTELEZO_SQL_FUTTASD.sql

Feltöltés után:
1. GitHub deploy lefut
2. Oldalon Ctrl+F5
3. Ha még régi fájl jön: F12 > Application > Service Workers > Unregister, majd Ctrl+F5
